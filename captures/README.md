# Capture storage

Store only sanitised example captures in this directory.

For every weighing capture, record:

- date and time
- scale model
- displayed weight
- displayed body fat, water, muscle, bone, BMI or BMR where shown
- selected user profile
- whether the official app was involved
- operating system and Bluetooth adapter
- raw packets with characteristic UUID and timestamp

Keep unredacted personal health captures under `captures/private/`, which is excluded by `.gitignore`.
